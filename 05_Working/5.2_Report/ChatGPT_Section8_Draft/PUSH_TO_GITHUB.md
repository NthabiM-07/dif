# Push instructions

From a terminal:

```bash
git clone https://github.com/NthabiM-07/dif.git
cd dif
cp -R /path/to/INGB417_PROJECT_REPO_READY ./INGB417_PROJECT

git add INGB417_PROJECT
git commit -m "Add INGB417 DigMore facility design project"
git push origin master
```

If you are using Windows PowerShell:

```powershell
git clone https://github.com/NthabiM-07/dif.git
cd dif
Copy-Item -Recurse "C:\path\to\INGB417_PROJECT_REPO_READY" ".\INGB417_PROJECT"

git add INGB417_PROJECT
git commit -m "Add INGB417 DigMore facility design project"
git push origin master
```
